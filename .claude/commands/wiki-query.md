# SKILL: wiki-query

사용자 질문에 대해 위키 지식 베이스를 검색하고 답변합니다.

## 실행 전 체크

1. `RULES.md` 읽기 (이미 읽었다면 생략)
2. `index.md` 읽기 (페이지 목록 파악)

## 절차

**Step 1 — 키워드 추출**
사용자 질문에서 핵심 키워드 2-3개 추출

**Step 2 — index.md 탐색**
- index.md의 태그·제목·요약에서 키워드 매칭
- 관련 슬러그 목록 작성

**Step 3 — wiki 페이지 읽기**
- 관련 슬러그의 `wiki/NN-slug.md` 파일 읽기
- related 필드의 연관 페이지도 추가 탐색

**Step 4 — 답변 생성**
```
답변 형식:
[핵심 답변 — 2-3문장]

**근거 페이지:**
- [[slug1]]: 관련 내용 요약
- [[slug2]]: 관련 내용 요약

**추가 참고:**
- [[연관slug]]: 더 자세한 내용
```

**Step 5 — journal 기록**
```
- [YYYY-MM-DD HH:MM] Query: "<질문>" — wiki/NN-slug.md 참조
```

## 페이지를 찾지 못한 경우

```
"해당 개념은 아직 위키에 없습니다.
 관련 raw/ 파일이 있다면 /wiki-ingest로 추가할 수 있습니다."
```

## 답변 규칙

- 추측하지 말고 위키에 있는 내용만 인용
- 출처를 반드시 명시 (`Source: [[slug]]`)
- 한국어 질문 → 한국어 답변
